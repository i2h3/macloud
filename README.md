# Macloud

A minimalistic menu bar extra for macOS which controls Docker Desktop to quickly deploy an ephemeral Nextcloud server container for client development.

## Who is this for?

For developers working on client apps which need to connect to a Nextcloud server.

## Why?

Because I am lazy and rather click twice than type dozens of characters.
Even pulling it out of my snippet manager is too much for me.
